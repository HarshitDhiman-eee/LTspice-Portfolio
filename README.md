# LTspice Portfolio

Welcome to my LTspice Portfolio!

This repository contains a collection of circuit simulations created using LTspice as part of my learning journey in Electrical Engineering. The objective of this repository is to strengthen my understanding of analog electronics, circuit analysis, and electronic system design through practical simulations.

## Contents

This repository will gradually include:

- Basic diode circuits
- Rectifiers (Half-Wave, Full-Wave, Bridge)
- Clippers and Clampers
- RC, RL and RLC circuits
- BJT transistor circuits
- Operational Amplifier (Op-Amp) applications
- Active filters
- Oscillators
- Power electronics circuits
- Mini engineering projects

Each simulation includes:
- LTspice schematic (.asc)
- Circuit screenshot
- Output waveform
- Brief explanation of the circuit and observations

## Purpose

The aim of this repository is to document my progress, improve my circuit simulation skills, and build a practical portfolio that complements my Electrical Engineering coursework.

## Tools Used

- LTspice
- Electrical Engineering Fundamentals
- Circuit Analysis
